# Road-Accident-Dashboard-
An interactive Road Accident Analysis Dashboard built using Microsoft Excel to analyze accident trends, casualties, vehicle types, location-wise impact, and time-based patterns. This project uses real-world data to generate meaningful insights through Pivot Tables, Charts, Slicers, and Dynamic KPIs.
